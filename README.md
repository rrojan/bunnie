# Bunnie

Implementing ExpressJS from scratch, for Typescript & Bun

## Usage

To install dependencies run:

```bash
bun install
```

Run the server:

```bash
bun run main.ts
```

This project was created using `bun init` in bun v1.2.2. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
